# arcgis-enterprise-clone-content Python Toolbox (`arcgis_cloning.pyt`)

This is the documentation to be included if planning on packaging the `arcgis_cloning.pyt`
toolbox using the `make pyt_pkg` command invoking the `./scripts/make_pyt_archive.pyt`. This script
packages the `arcgis_cloning.pyt`, supporting Python package 
`./src/arcgis_cloning` and any dependencies listed in `pyproject.toml`. This enables you to
distribute the Python toolbox for non-technical users with any custom libraries you are utilizing.
